[postgreSQL 9.4 설치]

1. yum repository 등록
yum install http://yum.postgresql.org/9.4/redhat/rhel-5-x86_64/pgdg-centos94-9.4-1.noarch.rpm
또는 
wget http://yum.postgresql.org/9.4/redhat/rhel-5-x86_64/pgdg-centos94-9.4-1.noarch.rpm
 rpm -ivh pgdg-centos94-9.4-1.noarch.rpm

2. postgresSQL 설치
yum install postgresql94 postgresql94-devel postgresql94-contrib postgresql94-server

yum install postgresql94
 yum install postgresql94-devel 
 yum install postgresql94-contrib
 yum install postgresql94-server

3. 계정은 아래의 경로로 자동 생성 되는 군요...
 .##useradd postgres -d /var/lib/pgsql 
비밀번호 설정
passwd postgres

4. 환경 설정
chown -R postgres.postgres /var/lib/pgsql

su - postgres
 vi .bash_profile

export PATH="/usr/pgsql-9.4/bin:$PATH"
 export POSTGRES_HOME=/usr/pgsql-9.4
 export PGLIB=$POSTGRES_HOME/lib
 export PGDATA=/var/lib/pgsql/9.4/data 

5. DB Initialize
주의 사항...... /var/lib/pgsql/9.4/data 아래 내용을 깨끗하게 지워라
/etc/init.d/postgresql-9.4 initdb -E 'UTF-8' -D /var/lib/pgsql/9.4/data

6. 외부접근 설정
vi /var/lib/pgsql/9.4/data/postgresql.conf
 listen_addresses = '*'
 port = 5432
 max_connections = 100

vi /DATA/airelive/pgsql/data/pg_hba.conf 
 host all all 0.0.0.0/0 md5 

7. PostgreSQL service 등록, 시작
chkconfig --add /etc/init.d/postgresql-9.4
 chkconfig postgresql-9.4 on

service postgresql-9.4 start

8. DB user 생성
su - postgress
 createuser -P jirauser
 ua0919
 createdb jiradb

9. pgadmin 으로 확인
( http://www.pgadmin.org/ )
